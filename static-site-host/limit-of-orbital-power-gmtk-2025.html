Limit of Orbital Power (GMTK 2025)

Limit of Orbital Power is a video game which a few friends and I made for the 2025 GMTK Game Jam.

The GMTK Game Jam is an annual competition to design and make a video game in 4 days based on a theme that's only revealed when the game jam starts. This year's theme was "Loop".

This was my first time ever participating in a game jam. In addition to myself, our team included Gio Zavalza on programming, Alex Montgomery on 3D art, and Ian Montgomery on music and SFX. Everyone did a great job and I think, given the time constraints, this turned out nice. I supported the project with programming and also 2D art.

The goal of the game is to make the ball that you control reach a speed which crashes the game. It plays a little bit like Burrito Bison but on a circular track. The project was done in Unity.

If you want to try it out, check out the Limit of Orbital Power itch.io page.
